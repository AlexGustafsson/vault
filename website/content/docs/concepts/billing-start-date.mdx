# Billing start date

This is the billing start date associated with the reporting cluster. This will be the license start date if not configured.

As of 1.15.13, 1.16.7, 1.17.3, 1.18+, Vault clusters will automatically adjust the billing start date to provide a year's worth of utilization data.
