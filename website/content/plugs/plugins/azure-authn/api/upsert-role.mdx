# UpsertRole endpoint

Update or register a role with the plugin. Role types have specific entities
that can perform login operations against the plugin and the specified
constraints apply to those authenticated entities.

| Method | Path                                      |
| ------ | ----------------------------------------- |
| `POST` | `/{mount_path}/{plugin_name}/role/{name}` |
