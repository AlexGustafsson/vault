**`-help` | `-h`**

Print the in-tool help menu for the CLI command associated with your Vault
version. In-tool help displays the available subcommands, parameters, and
flags for the CLI command.

```shell-session
$  vault print --help

Usage: vault print <subcommand>

  Group subcommands that interact with Vault runtime values.

Subcommands:

  token    Print the current Vault token
```
