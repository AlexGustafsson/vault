**`-output-curl-string`**

Print the CLI command as a `curl` command against the Vault API instead of
running the command.

```shell-session
$ vault write               \
    -output-curl-string     \
    auth/userpass/users/bob \
    password="long-password"

curl -X PUT -H "X-Vault-Request: true" -H "X-Vault-Token: $(vault print token)" -d '{"password":"long-password"}' http://127.0.0.1:8200/v1/auth/userpass/users/bob
```
