<a id="environment-request" />

**`environment`** `<enum: "AzurePublicCloud">` <b style={{color: 'var(--gray-3)'}}>optional</b>

<sup>Optional Vault environment variable: <code>AZURE_ENVIRONMENT</code></sup>

String indicating the target Azure cloud environment:

- AzurePublicCloud - tenants in the Azure Public Cloud
- AzureUSGovernmentCloud - tenants in the Azure US Government Cloud
- AzureChinaCloud - tenants in the Azure China Cloud
- AzureGermanCloud - tenants in the Azure German Cloud
