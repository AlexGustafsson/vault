**`identity_token_audience`** `<string>: ""` <b style={{color: 'var(--gray-3)'}}>optional</b> <EnterpriseAlert inline />

Audience claim value for Azure plugin identity tokens. The audience must match
the allowed audiences configured for the target [Federated Identity Credential].
Mutually exclusive with `client_secret`.
