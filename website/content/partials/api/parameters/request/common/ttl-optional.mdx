**`ttl`** `<string>: ""` <b style={{color: 'var(--gray-3)'}}>optional</b>

The time to live (TTL) in [duration format](/vault/docs/concepts/duration-format)
for the object (e.g., tokens).

When `ttl` is unset, Vault assigned a default time to live based on any
applicable leases or policies.
