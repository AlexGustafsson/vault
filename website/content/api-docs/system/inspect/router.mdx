# `/sys/internal/inspect/router`
The `/sys/internal/inspect/router` is intended to inspect a specific internal subsystem for debugging purposes.
It can be accessed with a root token or sudo privileges.

## Root

This endpoint returns a list of router entries in the router's root tree.

| Method | Path          |
| :----- | :------------ |
| `GET`  | `/sys/internal/inspect/router/root` |

### Sample Request

```shell-session
$ curl \
    --header "X-Vault-Token: ..." \
    http://127.0.0.1:8200/v1/sys/internal/inspect/router/root
```
