# `/sys/mounts-all`

The `/sys/mounts-all` endpoint is used to manage mount listing
visibility. The response generated by this endpoint is based on the
`listing_visibility` value on the mount, which can be set during mount time or
via mount tuning.

## Get Available Visible Mounts

This endpoint lists all enabled auth methods.

| Method | Path                      |
| :----- | :------------------------ |
| `GET`  | `/sys/mounts-all` |

### Sample Request

```shell-session
$ curl \
    http://127.0.0.1:8200/v1/sys/mounts
```

### Sample Response

```json
{
  "auth": {
    "github/": {
      "description": "GitHub auth",
      "type": "github"
    }
  },
  "secret": {
    "custom-secrets/": {
      "description": "Custom secrets",
      "type": "kv"
    }
  }
}
```
